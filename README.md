# 🎯 QASonar

> Um mini game educativo para testes de QA com visual de sonar radar! 📡

Transforme seu aprendizado de QA em uma aventura nautical! Navegue pelas ondas do conhecimento e descubra o que anda escondido no profundo do universo de testes de software. 🦑

![Preview do QASonar](https://via.placeholder.com/800x400?text=QASonar+Preview)

---

## 🚀 O que tem no pacote?

- **Sonar animated** - 3 pontos的神秘物品 para você clicar e descobrir
- **3 sessões de perguntas**:
  - 🧠 Fundamentos de QA
  - 🔌 Testes de API
  - 🤖 Automação com Cypress
- **5 perguntas por sessão** - Prepare o cinto!
- **Feedback imediato** - Saberá na hora se acertou ou errou (sem suspense!)
- **Resultados com explicações** - Aprenda enquanto joga

## ⚡ Como rodar o bicho

```bash
# Instala as dependências (não pule essa parte!)
npm install

# Bora lá! 🚀
npm run dev
```

Abra o navegador em `http://localhost:5173` e boa viagem!
